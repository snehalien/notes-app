# 📝 Notes App

A feature-rich Notes App built using HTML, CSS, and JavaScript with user authentication and persistent storage.

---

## 🚀 Features

- 🔐 User Registration & Login (localStorage-based)
- 📂 Sidebar displaying all notes
- ✏️ Add and view notes
- 🔍 Search notes in real-time
- 🎨 Color-coded notes
- 🌙 Dark mode toggle
- 💾 Notes saved per user (persistent after refresh)

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- localStorage (for data persistence)

---

## 📌 Description

This project is a multi-user notes application that allows users to create, manage, and search notes efficiently. Each user has their own stored notes using browser localStorage.

The application demonstrates core frontend concepts such as:
- CRUD operations
- DOM manipulation
- State management
- User authentication simulation

---

## 🎯 Purpose

This project was built to:
- Practice frontend development
- Understand how web apps manage user data
- Simulate real-world application behavior without a backend

---

## 📁 How to Run

1. Download or clone the repository
2. Open `login.html` in your browser
3. Register a new user
4. Login and start adding notes

---

## ⚠️ Note

This project uses localStorage for authentication and data storage.  
It is intended for learning purposes and is not secure for real-world applications.
